Welcome to the Cinema Room Application!

This application allows you to view and purchase seats for movies in our cinema.

Features include:

  • View available seats
    • Green buttons → available seat
    • Red buttons → seat taken
  • Purchase tickets
    • To buy a seat, click on the green free seat of your choice
  • Return tickets
    • Enter the Ticket ID in the provided format: row_column (e.g., 1_1 for row 1 and column 1)
  • Admin statistics (login required)
    • Enter admin as username and password to log in to the statistics page

Use the navigation bar above to get started.
